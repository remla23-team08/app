# App

### Requirements
- NodeJS
- NPM

### Installing dependencies
`npm install`

### Building
`npm run build`

### Running
`npm run start`

### Formatting code
`npm run format`

### UI Endpoint
`http://localhost:8083`

### Docker Build
`docker build -t <tag> .`

### Docker Run
`docker run --rm -p8083:8083 <tag>`
